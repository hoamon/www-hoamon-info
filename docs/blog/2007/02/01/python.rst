Python: 可變物件
================================================================================

在 Python 中，幾乎每樣東西都是物件。而可變物件的意思即允許使用者對物件作新增、刪除、修改的動作。

.. more::

可變物件常見的有 list 及 dictionary(hash)。

不可變物件常見的是 tuple 。
